# BDKS Frontend

보루시아 도르트문트 코리아 팬들을위한 실시간 평점입력 사이트

## 사용 가능한 Commands

코드 작성/수정 시 아래 commands를 활용하세요:

| Command            | 설명                                                                                                           |
| ------------------ | -------------------------------------------------------------------------------------------------------------- |
| `/bdks-convention` | **개발 컨벤션** (정보의 원천) — 프로젝트 구조, 네이밍, 컴포넌트 패턴, API 패턴, UI 패턴, 테스팅 등 모든 컨벤션 |
| `/yd-ui`           | **yd-ui 디자인 시스템** — 디자인 토큰(색상/타이포/간격) 및 컴포넌트 문서 참조                                  |
| `/bdks-3d-icon`    | **3D 아이콘 프롬프트 생성기** — BDKS UI용 3D 스타일 아이콘 이미지 생성 프롬프트                                |
